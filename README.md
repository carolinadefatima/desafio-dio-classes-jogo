# Desafio de Projeto: Escrevendo as Classes de um Jogo

O contém a resolução de um desafio focado em **Programação Orientada a Objetos (POO)** utilizando a linguagem **Java**. O objetivo foi criar uma classe genérica para representar heróis de uma aventura com diferentes tipos e ataques.

## Objetivo do Desafio

Criar uma classe que represente um herói com propriedades específicas e um método de ataque que varie conforme o tipo do personagem:

- **Propriedades:** Nome, Idade e Tipo.
- **Tipos Disponíveis:** Guerreiro, Mago, Monge e Ninja.
- **Lógica de Ataque:**
  - **Mago** -> usou magia
  - **Guerreiro** -> usou espada
  - **Monge** -> usou artes marciais
  - **Ninja** -> usou shuriken
